# Milestone Plan

## 1. MVP (Minimum Viable Product)
- **기본 게임 로직 구현**
- **테마/난이도 선택 기능**
- **이벤트 로그 시스템**
- **UI/UX 개선**
- **시뮬레이터/실제 디바이스 테스트**
- **문서화 (이슈, 워크로그, 커밋 메시지)**

## 2. 게임 결과 리포트 & 로그 저장
- **게임 결과 리포트 UI 추가**
- **이벤트 로그 파일 저장 기능**

## 3. 테스트 자동화 & CI/CD
- **UI 테스트, 유닛 테스트 추가**
- **CI/CD 파이프라인 구축**

## 4. 추가 피처 개발
- **난이도 확장**
- **테마 추가**
- **게임 기록/랭킹 기능**

## 5. 문서화 최신화
- **BRD, SOW, FSD, Low-level Design, ERD, Runbook 등 최신화**
